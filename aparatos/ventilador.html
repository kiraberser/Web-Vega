Refacción de Ventiladores
Campo
El campo puede referirse a la parte del motor de un ventilador que genera el movimiento rotativo necesario para hacer girar las aspas.
Aspa
Las aspas son las partes móviles de un ventilador que generan el flujo de aire al girar.
Bujes
Los bujes son los elementos que permiten que las aspas de un ventilador giren suavemente alrededor del eje central.
Rotor
El rotor es la parte del motor de un ventilador que, junto con el campo, genera el movimiento rotativo necesario para hacer girar las aspas.
Fusible
El fusible es un componente de seguridad utilizado en el circuito eléctrico de un ventilador para protegerlo contra sobrecargas eléctricas.
